# Fast-Replicate 项目需求列表

## 项目概述

创建一个高度可复用的 Next.js 模板，用于快速搭建调用 Replicate.com AI 模型的 SaaS 应用。采用订阅付费模式，用户通过购买会员获得 AI 模型使用权限。

重点要求：页面语言使用英语。
